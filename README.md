
# Visualisasi Data Pengguna Media Sosial di Asia 2024

**Pembuat:** Harry Zulkifli (NIM: 220660121081)

## Deskripsi
Proyek ini menampilkan visualisasi interaktif data pengguna media sosial di beberapa negara Asia pada periode 2015–2024. Terdapat beberapa jenis visualisasi: Line, Bar, Pie, Histogram, Scatter, dan Peta Choropleth (Asia).

## Cara Menjalankan
1. Ekstrak folder.
2. Buka `index.html` di browser (Chrome/Edge disarankan). Koneksi internet diperlukan untuk memuat Chart.js dan Google GeoChart CDN.
3. Gunakan menu di atas untuk berpindah antar visualisasi.

## Teknologi
- HTML, CSS, JavaScript
- Chart.js (grafik)
- Google GeoChart (peta choropleth)
